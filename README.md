# Veerzone Stream Overlay

A lightweight public HTML overlay for StreamChamp / browser source use.

## Main public URL after GitHub Pages is enabled

```text
https://malleshpithani-hub.github.io/veerzone-overlay/
```

## How to enable GitHub Pages

1. Open this repository on GitHub.
2. Go to Settings.
3. Open Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Branch: main.
6. Folder: /root.
7. Save.
8. Wait until GitHub shows the published site URL.

## StreamChamp setup

1. Copy the public GitHub Pages URL.
2. Open StreamChamp.
3. Add a browser / webpage overlay source.
4. Paste the URL.
5. Set width and height according to your stream canvas.

## Custom URL examples

Default:

```text
https://malleshpithani-hub.github.io/veerzone-overlay/
```

Custom BO2 event:

```text
https://malleshpithani-hub.github.io/veerzone-overlay/?match=BO2%20SCRIMS&time=10:30%20PM&slots=3%20LEFT&total=₹600&winner=Winner%20₹350&runner=Runner%20₹150&frag=Top%20Fraggers%20Rewards
```

Registration live:

```text
https://malleshpithani-hub.github.io/veerzone-overlay/?status=REGISTRATION%20LIVE&message=BOOK%20YOUR%20SLOT&cta=REGISTER%20NOW%20%E2%80%94%20IT'S%20FREE
```
